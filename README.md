Этот репозиторий содержит реализацию классического алгоритма «Шифр Цезаря» на языке C++ для защиты текстовых паролей. Программа написана с нуля, без использования сторонних криптографических библиотек.

## Пояснения по выполнению задачи

В рамках практического задания был написан скрипт, который шифрует введенный пользователем пароль путем сдвига символов латинского алфавита на заданное число позиций (`shift_key = 5`). Также реализована функция обратной расшифровки. Алгоритм обрабатывает как строчные, так и заглавные буквы, а цифры и спецсимволы оставляет без изменений.

**Детальный разбор работы алгоритма на примере:**

1. **Изначальный пароль:** В качестве тестового пароля использована строка `Roblox3D`.
2. **Как он зашифровался:** * Программа посимвольно читает строку и применяет сдвиг `+5` к каждой букве.
   * Заглавная 'R' сдвигается по алфавиту на 5 шагов вправо и становится 'W'.
   * Строчные 'o' -> 't', 'b' -> 'g', 'l' -> 'q', 'o' -> 't', 'x' -> 'c'.
   * Цифра '3' игнорируется математикой сдвига и переносится в итоговую строку как есть.
   * Заглавная 'D' -> 'I'.
   * **Результат шифрования:** Пароль преобразуется в строку `Wtgtqc3I`.
3. **Как происходит расшифровка:** * Для расшифровки вызывается та же функция шифрования, но с высчитанным обратным сдвигом по формуле `26 - (shift % 26)`. При исходном сдвиге `5`, обратный шаг равен `26 - 5 = 21`.
   * Смещение зашифрованных букв (`Wtgtqc3I`) на 21 позицию вправо по замкнутому алфавитному кругу из 26 букв возвращает символы на их исходные места.
   * **Результат расшифровки:** Мы снова получаем изначальный пароль `Roblox3D`.

## Инструкция по запуску
1. Скомпилируйте исходный код с помощью компилятора C++ (например, g++ в консоли):
   `g++ main.cpp -o cipher`
2. Запустите скомпилированный файл:
   `./cipher` (для Linux/macOS) или `cipher.exe` (для Windows).
